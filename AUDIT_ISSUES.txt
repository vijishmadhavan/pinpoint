         compress_pdf, add_page_numbers, pdf_to_word (all PyMuPDF usages)

#4  Resource leak: PDF OCR                   extractors.py:161-218
    Status: FIXED

    Fix: try/finally for doc (initial open) and doc_render (OCR rendering)

#5  No transaction rollback in batch_move    api/files.py:603-641
    Status: FIXED
    Was: Single conn.commit() after all moves — partial failure = inconsistent state
    Fix: conn.commit() per file after successful move, conn.rollback() on DB error

#6  Unbounded memory: sessionCosts           bot/index.js:105
    Status: FIXED
    Was: Object grows forever, never GC'd per chatJid
    Fix: Cleaned in resetSession() + periodic GC (every 15min, 2hr stale threshold)

#7  Unbounded memory: actionLedger           bot/index.js:112
    Status: FIXED

    Fix: Same as #6 — cleaned in resetSession() + periodic GC

#8  Unbounded memory: TempStore              bot/index.js:396
